## Claude Desktop Install

### MCP Server (17 Fabric tools)

1. Download `fabric-cli-mcp.mcpb`
2. Double-click to open in Claude Desktop
3. Click **Install**
4. *(To remove)* Settings > Extensions > Fabric CLI (...) > Uninstall

Requires: [uv](https://docs.astral.sh/uv/), [fab CLI](https://pypi.org/project/fabric-cli/)

### Skill (Fabric CLI guidance)

1. Download `fabric-cli-skill.zip`
2. Open Claude Desktop > **Settings** > **Skills**
3. Click **Upload skill** > select the ZIP
4. *(To remove)* Settings > Skills > fabric-cli (...) > Delete

Requires: Pro, Max, Team, or Enterprise plan
